# Aya’s App

A sweet toddler-friendly web app for ages 2–5.

## Included in v1

- Rainbow musical keyboard
- Bubble popping game
- Numbers and counting game
- Safe cartoon watching section using legal public-domain sources
- Big colorful UI designed for small children

## Tech

- Vite
- Vanilla JavaScript
- CSS

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
